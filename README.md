# 个人名片 Cards

基于 Vercel Severless 和 node.js 开发.

访问地址：[cards.jerryz.com.cn/api](https://cards.jerryz.com.cn/api)

推荐使用生成器：[cards.jerryz.com.cn](https://cards.jerryz.com.cn)

![](https://card.jerryz.com.cn/white)
